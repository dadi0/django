cgi(Common Gateway Interface)通用网关接口
定义客户端与Web服务器的交流方式
wsgi(Python Web Server Gateway Interface)Web服务器网关接口
基于python以cgi为标准做的扩展
基于HTTP协议，不支持WebSocket
asgi(Asynchronous Server Gateway Interface)异步网关接口
是wsgi的扩展，支持WebSocket

Object-relational mapping(ORM)对象关系映射
使用面像对象的编程语言在不兼容类型系统之间转换数据的编程技术

一个类对应一张表,一个实例对应表中的一条数据,方法对应sql语句

字典采用hash表的实现方式,为了减少散列冲突发生概率,散列表必须是稀疏的,所以字典在内存上开销大
列表采用动态数组,列表存储的是每个元素在内存中的地址(引用),当列表空白占位低于1/3时,
会在内存中开辟一块更大的空间,并将旧列表中存储的地址复制到新列表中
